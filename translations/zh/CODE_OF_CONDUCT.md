# 行为准则

## 我们的承诺

我们致力于为所有人提供一个友好和包容的社区，无论其背景、身份或经验水平。

## 标准

**期望行为：** 使用友好和包容的语言，尊重不同的观点和经验，优雅地接受建设性批评。

**不可接受的行为：** 骚扰、恶意攻击、人身攻击、发布他人私人信息。

## 执行

社区领导者将对违规行为采取适当措施。

改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1。
