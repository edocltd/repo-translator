# Política de Segurança

## Escopo

repo-translator processa arquivos markdown e gera prompts de tradução. Não executa código arbitrário, não faz solicitações de rede e não manipula credenciais.

## Reportar uma vulnerabilidade

1. **NÃO** abra um issue público
2. Use [GitHub Private Vulnerability Reporting](https://github.com/edocltd/repo-translator/security/advisories)

Confirmaremos o recebimento em 48 horas.
