# Código de Conduta

## Nosso compromisso

Comprometemo-nos a fornecer uma comunidade acolhedora e inclusiva para todos.

## Padrões

**Comportamento esperado:** Linguagem acolhedora, respeito por diferentes pontos de vista, aceitação de críticas construtivas.

**Comportamento inaceitável:** Assédio, trolling, ataques pessoais, publicação de informações privadas.

## Aplicação

Líderes da comunidade tomarão medidas apropriadas em resposta a violações.
