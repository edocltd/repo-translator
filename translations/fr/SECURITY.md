# Politique de sécurité

## Portée

repo-translator traite des fichiers markdown et génère des prompts de traduction. Il n'exécute pas de code arbitraire, ne fait pas de requêtes réseau et ne gère pas de credentials.

## Signaler une vulnérabilité

1. **NE PAS** ouvrir un issue public
2. Utiliser [GitHub Private Vulnerability Reporting](https://github.com/edocltd/repo-translator/security/advisories)

Nous accuserons réception dans les 48 heures.
