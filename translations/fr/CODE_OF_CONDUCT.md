# Code de conduite

## Notre engagement

Nous nous engageons à fournir une communauté accueillante et inclusive pour tous.

## Standards

**Comportement attendu :** Langage accueillant, respect des différents points de vue, acceptation des critiques constructives.

**Comportement inacceptable :** Harcèlement, trolling, attaques personnelles, publication d'informations privées.

## Application

Les responsables prendront les mesures appropriées en réponse aux violations.
