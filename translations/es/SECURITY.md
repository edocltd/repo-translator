# Política de Seguridad

## Alcance

repo-translator procesa archivos markdown y genera prompts de traducción. No ejecuta código arbitrario, no realiza solicitudes de red ni maneja credenciales de autenticación.

## Reportar una vulnerabilidad

Si descubres un problema de seguridad, repórtalo responsablemente:

1. **NO** abras un issue público
2. Usa [GitHub Private Vulnerability Reporting](https://github.com/edocltd/repo-translator/security/advisories)

Confirmaremos la recepción dentro de 48 horas.
