# セキュリティポリシー

## 範囲

repo-translatorはmarkdownファイルを処理し、翻訳プロンプトを生成します。任意のコード実行、ネットワークリクエスト、認証情報の処理は行いません。

## 脆弱性の報告

1. 公開issueを**開かないでください**
2. [GitHub Private Vulnerability Reporting](https://github.com/edocltd/repo-translator/security/advisories)を使用してください

48時間以内に受領を確認します。
