# 行動規範

## 私たちの約束

背景、アイデンティティ、経験レベルに関係なく、すべての人にとって歓迎的で包括的なコミュニティを提供することを約束します。

## 基準

**期待される行動：** 歓迎的で包括的な言葉遣い、異なる視点と経験の尊重、建設的な批判の受け入れ。

**許容されない行動：** ハラスメント、荒らし、個人攻撃、他者の個人情報の公開。

## 執行

コミュニティリーダーは違反に対して適切な措置を講じます。

## 帰属

[Contributor Covenant](https://www.contributor-covenant.org/) バージョン2.1より改編。
