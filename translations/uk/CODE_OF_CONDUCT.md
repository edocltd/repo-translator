# Кодекс поведінки

## Наше зобов'язання

Ми зобов'язуємось забезпечити привітну та інклюзивну спільноту для всіх, незалежно від походження, ідентичності чи рівня досвіду.

## Стандарти

**Очікувана поведінка:**
- Використовуйте привітну та інклюзивну мову
- Поважайте різні точки зору та досвід
- Приймайте конструктивну критику з гідністю
- Зосереджуйтесь на тому, що найкраще для спільноти

**Неприйнятна поведінка:**
- Переслідування, тролінг або особисті атаки
- Публікація приватної інформації інших
- Дискримінаційна мова або зображення
- Будь-яка поведінка, яку можна обґрунтовано вважати неприйнятною

## Застосування

Лідери спільноти вживатимуть відповідних заходів у відповідь на порушення. Повідомлення можна надсилати через [GitHub Issues](https://github.com/edocltd/repo-translator/issues) або зв'язавшись з мейнтейнером напряму.
