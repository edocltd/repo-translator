# Політика безпеки

## Область

repo-translator обробляє markdown-файли та генерує промпти для перекладу. Він не виконує довільний код, не здійснює мережевих запитів та не обробляє облікові дані.

## Повідомлення про вразливість

Якщо ви виявили проблему безпеки, повідомте відповідально:

1. **НЕ** відкривайте публічний issue
2. Використовуйте [GitHub Private Vulnerability Reporting](https://github.com/edocltd/repo-translator/security/advisories)
3. Або зв'яжіться з мейнтейнером напряму

Ми підтвердимо отримання протягом 48 годин та надамо терміни для виправлення.
