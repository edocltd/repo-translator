# Sicherheitsrichtlinie

## Umfang

repo-translator verarbeitet Markdown-Dateien und generiert Übersetzungs-Prompts. Es führt keinen beliebigen Code aus, stellt keine Netzwerkanfragen und verarbeitet keine Anmeldedaten.

## Eine Schwachstelle melden

1. **KEIN** öffentliches Issue eröffnen
2. [GitHub Private Vulnerability Reporting](https://github.com/edocltd/repo-translator/security/advisories) verwenden

Wir bestätigen den Eingang innerhalb von 48 Stunden.
